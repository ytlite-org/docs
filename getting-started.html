How to Use YTLite

Self-Hosting

  1. Clone the YTLite repository
  2. Install Python dependencies
  3. Run the Flask server
  4. Access through your browser
git clone https://github.com/ytlite-org/YTLite.git
cd YTLite
pip install -r requirements.txt
python app.py

Public Instance

If a public instance is available, simply visit the URL in your browser.

Basic Navigation

  • Search using the search box
  • Click on videos to watch
  • Browse trending content on the homepage
  • View channel pages by clicking channel names
